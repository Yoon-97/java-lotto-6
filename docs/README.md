```

[13, 14, 16, 38, 42, 45]

[2, 13, 22, 32, 38, 45] 
[1, 3, 5, 14, 22, 45]

4개 일치 (50,000원) - 0개
```

## 기능 구현 목록
- 로또 번호 생성(1~45)
- 1개의 로또에 중복되는 숫자 있는지 체크
- 금액에 맞는 로또 구입 갯수 구하기
- 6개의 중복되지 않은 번호 입력
- 보너스 번호 입력
- 등수 계산
- 당첨 내역 출력
- 수익률 출력
---

## 예외 사항
- 로또 번호가 6개 이상인 경우
- 로또 번호가  1~45가 아닌 경우
- 구입금액이 1000원 단위로 끊어지지 않는 경우
